# How to Run — Kotlin JVM Code Generation

## Prerequisites

- Swift 6.2+
- macOS 13+ or Linux

## Build the Project

```bash
git clone https://github.com/Garnik645/swift-java.git
cd swift-java
swift build
```

## Prepare Your Swift Module

The tool requires a compiled Swift module interface. Given a Swift source file:

```swift
// MyModule.swift
public func add(a: Int, b: Int) -> Int {
    return a + b
}

public func greet(name: String) -> String {
    return "Hello, \(name)!"
}

public func isEven(n: Int) -> Bool {
    return n % 2 == 0
}
```

Compile it into a module interface:

```bash
swiftc MyModule.swift \
  -module-name MyModule \
  -emit-module-interface \
  -enable-library-evolution \
  -emit-module-path /tmp/MyModule.swiftmodule \
  -emit-library \
  -o /tmp/libMyModule.dylib
```

## Generate Kotlin Sources

```bash
swift run swift-java jextract \
  --input-swift /tmp \
  --swift-module MyModule \
  --output-swift /tmp/out-swift \
  --output-java /tmp/out-kotlin \
  --mode kotlin-jvm
```

## View the Output

```bash
cat /tmp/out-kotlin/MyModule.kt
```

Expected Kotlin output:

```kotlin
// Generated by swift-java
// Swift module: MyModule

object MyModule {
  init {
    System.loadLibrary("swiftCore")
    System.loadLibrary("MyModule")
  }

  private external fun `$add`(a: Long, b: Long): Long
  fun add(a: Long, b: Long): Long = `$add`(a, b)

  private external fun `$greet`(name: String): String
  fun greet(name: String): String = `$greet`(name)

  private external fun `$isEven`(n: Long): Boolean
  fun isEven(n: Long): Boolean = `$isEven`(n)
}
```

Expected Swift thunk output in `/tmp/out-swift/`:

```swift
@_cdecl("Java_MyModule__00024add__JJ")
public func Java_MyModule__00024add__JJ(...) -> jlong { ... }
```

## End-to-End Proof (kotlin-swift-test)

A minimal runnable example is provided in `kotlin-swift-test/`. It proves the generated Kotlin code actually calls into Swift at runtime.

### Prerequisites

- `kotlinc` (`brew install kotlin`)
- JDK 21+

### Run it

```bash
cd kotlin-swift-test

# Step 1: compile Swift to native library
swiftc Sources/MyModule/MyModule.swift Sources/MyModule/MyModuleJNI.swift \
  -module-name MyModule \
  -emit-library \
  -o libMyModule.dylib

# Step 2: compile Kotlin
kotlinc MyModule.kt Main.kt -include-runtime -d app.jar

# Step 3: run
java --enable-native-access=ALL-UNNAMED \
  -Djava.library.path=. \
  -jar app.jar
```

### Expected Output

```
add(2, 3) = 5
add(10, 20) = 30
isEven(4) = true
isEven(7) = false
DONE.
```

## Type Mappings

| Swift | Kotlin |
|-------|--------|
| `Int` | `Long` |
| `Int32` | `Int` |
| `Bool` | `Boolean` |
| `Double` | `Double` |
| `String` | `String` |
| `Void` | `Unit` |
